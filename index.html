          USER GUIDE

          SETTINGS

          • School Name: Enter your school's full name.
          • Periods: Select the number of periods (4 to 10) from the dropdown.
          • Year: Enter the academic year (e.g., 2025-26).

          SUBJECTS

          • Enter Subjects: Add subjects separated by commas (e.g., MALI, ENG, HIN).
          • Allotted Periods: Specify periods for each subject (e.g., ENG-5, HIN-3).
          • STD: Enter class names (e.g., 5A, 5B) or use "All" for all classes.
          • Click the button to save.

          TEACHERS

          • Teacher Name: Use short names for convenience (e.g., SSK, BAS).
          • Class Charge: Assign a class to a teacher (e.g., SSK - 5B).
          • IMPORTANT: The 1st period will automatically be assigned to the Class Teacher (e.g., SSK in 5B).
          • Classes Taken: For other classes, set Charge to "None", select the class/subject, click "Add", then "Save".

          PREFERRED PERIODS

          • Set custom rules for specific subjects or teachers.
          • Example: For 'SV' subject with Teacher 'SSK', set Rule Type to 'Allowed', Day to 'Friday', and Period to '7'.
          • Click "Save Rule" to apply the constraint.
